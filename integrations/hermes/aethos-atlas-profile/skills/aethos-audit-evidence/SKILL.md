# Aethos Audit Evidence

Use Aethos audit and ledger tools to explain decisions, approvals, corrections,
source documents, agent runs, and workflow status. Summaries must be safe for
users and must not expose hidden traces or raw logs.

For document lineage prompts, use the documents audit read pack. Explicitly
cover engagement, bill, invoice, journal, and Inbox decision evidence. For each
available document, mention source filename, linked business record, extraction
state, and review next. If a category is missing, say it is missing or not
linked and state what should be reviewed next.

For COSEC filing reminder prompts, use the COSEC reminders read pack. Include
upcoming filing date, missing evidence, billing impact, and approval before
sending.
